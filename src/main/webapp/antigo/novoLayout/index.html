Home


O Guia da Escola Ideal é uma ferramenta WEB que permite filtrar as escolas do Centro-Oeste de acordo com as características físicas e metodológicas que mais se adaptem a sua preferência. Todos os dados apresentados nesta ferramenta são dados levantados pelo último censo do INEP (Instituto Nacional de Estudos e Pesquisas Educacionais Anísio Teixeira).

A equipe de desenvolvimento desenvolveu esta ferramenta visando facilitar o acesso às informações disponibilizadas segundo a Lei de Acesso a Informação sobre a infra-estrutura das escolas do Centro-Oeste como por exemplo a disponibilidade de banheiro com acessibilidade a deficiêntes, quadra de esportes coberta, cozinha, etc.

Esse site oferece um meio para relacionar as informações de acordo com sua maior prioridade e apresenta uma lista com as escolas que mais se encaixam com à sua necessidade. Você terá a oportunidade de selecionar entre várias caracteristicas oferecidas pelo site, das quais são indispensáveis na escola em que você deseja se matricular.

Convido você a encontrar a escola que mais se encaixa a sua necessidade!

FAQ

1. Os dados disponibilizados são atualizados anualmente?
Não. Os dados deste site se baseiam em informações coletadas no ano de 2012.

2. Onde posso encontrar os dados em que se baseia o site?
Os dados foram retirados desta planilha

3. O que são dados abertos?

4. Como eu utilizo o site?
Utilize as diversas ferramentas de navegação para encontrar o conjunto de dados que você procura. Você pode procurar escolas através de seu nome, estado e município.
